        PORT=3333
        HOST=0.0.0.0
        NODE_ENV=development
        APP_KEY=XVB9_yM91Egxd5AFN3hkzbUN0DGEZLq0
        DRIVE_DISK=local
        DB_CONNECTION=pg
        PG_HOST=localhost
        PG_PORT=5432
        PG_USER=lucid
        PG_PASSWORD=
        PG_DB_NAME=catShop
######################################################

obviamente substitua os campos user e password, com os dados do seu pgadm
